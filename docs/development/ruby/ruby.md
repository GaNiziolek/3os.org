# Ruby Cli Commands

## Get Gem & Ruby Environment Information

```bash
gem environment
```

## Update All the Gems

Install rubygems-update

```bash
gem install rubygems-update
```

Then run:

```bash
gem update --system
update_rubygems
```

## List All Installed Gems

```bash
ruby -S gem list --local
```
